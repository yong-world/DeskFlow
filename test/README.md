# 测试脚本说明

> 本目录包含项目的各类测试脚本（.bat 文件），双击即可运行并查看输出。

## 使用方法

1. 双击对应的 `.bat` 文件
2. 按照提示操作（如需要打开特定应用）
3. 查看测试输出结果

## 测试列表

### 阶段 0 - 技术验证

- **test_python_csharp_communication.bat** (调用 .ps1)
  - 测试内容：Python ↔ C# 子进程通信
  - 前置条件：已编译 C# 项目，已打开记事本
  - 预期结果：成功获取记事本的控件树
  - 说明：使用 PowerShell 脚本，支持中文显示和彩色输出

## 注意事项

- 首次运行前请确保已编译 C# 项目：`cd csharp && dotnet build`
- 部分测试需要特定应用处于运行状态（脚本会提示）
- 测试脚本会暂停等待查看结果，按任意键继续
